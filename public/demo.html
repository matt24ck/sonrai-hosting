Pretend storefront

Stands in for the live store so you can exercise the concierge widget locally.

The bubble (bottom-right) loads /embed.js with the selected data-store, pulls that store's real catalogue + theme tokens, and recommends real products. Switch stores with the dropdown (or add ?store=<id> to the URL).

Cart: this page isn't a Shopify storefront (no window.Shopify), so "Add" / "Add all" open the real Shopify cart permalink in a new tab — the fallback path. On the live store it switches automatically to in-page AJAX add-to-cart.

Sizing: sized items show size chips and keep "Add" disabled until a size is picked; one-size items add directly.
